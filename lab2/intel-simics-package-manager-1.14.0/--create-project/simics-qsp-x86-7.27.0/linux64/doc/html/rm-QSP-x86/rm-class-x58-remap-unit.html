x58-remap-unit

Description

The DMA/Interrupt remapping unit

Interfaces Implemented

conf_object, log_object, transaction_translator, translation_flush

Notifiers

cell-change
Notifier that is triggered after the object's cell was changed.
object-delete
Notifier that is triggered just before Simics object is deleted.
queue-change
Notifier that is triggered after the object's clock was changed. New clock can be obtained with the SIM_object_clock function.
state-change
Notifier on potential DML state change

Port Objects

bank.vtd
x58-remap-unit.vtd
port.HRESET
x58-remap-unit.HRESET
port.interrupt
x58-remap-unit.interrupt

Commands for this class

  • info – print information about the object
  • status – print status of the object

Attributes

apic_bus
Optional attribute; read/write access; type: o|[os]|n
APIC bus to handle interrupts

Required interfaces: apic_bus.

memory_space
Optional attribute; read/write access; type: o|[os]|n
Physical Memory
non_isoch
Optional attribute; read/write access; type: b
TRUE if this is the Non-Isochronous (i.e. first) VT-d engine on the X58
use_vtd_interrupt_decoding
Optional attribute; read/write access; type: b
Use VT-d address decoding to obtain handles when remapping interrupts, rather than x58 decoding
